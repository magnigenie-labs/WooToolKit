== Description ==
You have always been seeking for a One Stop Solution for all your woocommerce needs. 

Yes, with this plugin you can achieve remarkable scalability of functionalities and features, that too just under one plugin! The core speciality is that you can get support and  updates all in one place !

= FEATURES =

1.Accordions:

*  Converting your WooCommerce Tabs into Accordions with the help of powerful admin dashboard, it enables you to change each part of the accordion that matches your theme design and feel.

2.Catalog Mode:

* Catalog converts your store into catalog mode by hiding add to cart button, price tag, ratings, reviews.

3.Continue Shopping:

* Continue Shopping enables you to display continue shopping link to WooCommerce cart and thereby encouraging your customers to continue shopping without going to any other pages.

4.Custom Product Tabs:

* As the name suggests Custom Product Tabs add different custom tabs on each WooCommerce products.

5.Empty Cart:

* Empty Cart adds empty cart button to WooCommerce cart by clicking only once.

6.MailChimp:

* It enables you to capture emails of the orders on mailchimp. The plugin displays a checkbox on checkout page to achieve this.

7.Product Sorting Options:

* Product Sorting Options allows you to add new sorting options to WooCommerce shop page and also allows you to rename existing options.

8.Products Per Page:

* Products per page adds a dropdown to  your WooCommerce shop page which can be used by users to change the number of products displayed on the page.

9.Sequential  Order Numbers :

* Sequential Order Numbers enables you to have sequential numbers for the orders you get through your WooCommerce store.

10.WooCommerce Customiser:

* WooCommerce Customiser allows to customize default WooCommerce texts and features.

== Installation ==
1. Upload "Woo Toolkit" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Installation complete

= What actually is WooToolKit? =
WooToolKit is a scalable WooCommerce addon that enables you to beautifully manage all your WooCommerce functionalities and features from one place.

It will help you to have hassel free control on your WooCommerce addons.

= What does your WooToolKit contain? =
It contains 10 free kits to start with to help you in addressing your issues without encumbrances. With time, we are going to add more kits to help you even manage more complex features.
